# Spidey User Horoscope Function (Üye Burç Fonksiyonu)

A classic ASP/VBScript snippet developed by Yasin "Spidey" Koyutürk to calculate and display user horoscopes based on their birth dates.

## About This Project
This lightweight code snippet was designed for early community portals and forums, allowing webmasters to automatically display a user's zodiac sign with an accompanying icon.
